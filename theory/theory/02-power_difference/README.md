# 02 - パワー差がある場合の理論

単相性モデル $A_{ij}=p_i-p_j+v_i\times v_j$（[00/model.md](../00-single_compatibility_model/model.md)）を、これまでの**等パワー**（$p$ が相殺する状況）の絞り込みから**パワー差 $p_i-p_j\neq0$ 込み**へ拡張するトピック。model.md の予告「パワー差の補正ベクトルは別途扱う」の回収にあたる。

前提（単相性モデル・2択公式・構築ベクトル $V$）は [00-single_compatibility_model](../00-single_compatibility_model/) を、回転・回転平均は [01-rotating_team](../01-rotating_team/) を参照。

## 中心的な問い

1. **2択ゲームの値のパワー差込み一般化** — 利得 $A_{ij}=p_i-p_j+v_i\times v_j$ の 2×2 ゲームについて、ゲームの値と鞍点（実質一択）条件の場合分けを書き下す。パワー差 $\to 0$ の極限で [00/two_choice_game.md](../00-single_compatibility_model/two_choice_game.md) の結果に戻ることを確認する。
2. **パワー差の補正ベクトル** — パワー差はどんな条件で相性ベクトル側の変換（原点シフトなど）に吸収できるか。吸収できない場合に必要な最小の追加パラメータは何か。
3. **一択化の閾値** — どれだけパワー差があれば相性（外積項）を無視できるか。パワー差と相性項の大小で戦略構造（2択 ⇄ 実質一択）が切り替わる境界を明示する。

コアの3問が閉じたあとの発展として、構築ベクトル $V$ のパワー差込み一般化（二体構築は「1本のベクトル＋α」で表せるか）、01 の回転・回転平均のパワー差1次補正（摂動展開）、パワーと相性の交換レート（パワー1点＝相性どれだけ分か）を予定する。

## 記号

00・01 の記号をそのまま引き継ぐ。

- $p$：パワー、$v=(x,y)$：相性ベクトル、$v_i\times v_j=x_iy_j-y_ix_j$：外積（[00/model.md](../00-single_compatibility_model/model.md)）
- $V$：構築ベクトル（[00/team_vector.md](../00-single_compatibility_model/team_vector.md)）
- $g$：ゲームの値、$S$：長さの和、$\eta$：広がり効率（[01-rotating_team](../01-rotating_team/)）

02 で新しく導入する記号は、初出のファイルで定義した上でここに追記する。

- $\bar p_i,\bar p_j$：チーム平均パワー、$P=\bar p_i-\bar p_j$：チーム間の平均パワー差（[two_choice_game.md](two_choice_game.md)）
- $\Delta p_i=p_1-p_2,\ \Delta p_j=p_3-p_4$：チーム内パワー差（同上）
- $\bar v_i,\bar v_j$：相性ベクトルのチーム平均、$W=(v_1-v_2)\times(v_3-v_4)$：符号付き窓幅（＝混合の分母）、$m_i,m_j$：窓の中心（同上）

## 要点

- **チーム間の平均パワー差 $P$ は値に直行する**：$g=P+g|_{P=0}$ で、2択⇄一択の構造にも混合比にも一切効かない。戦略構造に効くのはチーム内パワー差 $\Delta p_i,\Delta p_j$ の2つだけ。
- **2択はチーム内パワー差の「窓」の内側でだけ生き残る**：各 $\Delta p$ が中心 $m$、共通幅 $|W|$ の開区間に入ることが本質的2択の条件。窓の端で混合比がちょうど $0/1$ に飽和し、実質一択へ連続に移行する。
- 2択領域の値は閉形式 $g=P+\dfrac{(v_1\times v_2)(v_3\times v_4)-m_im_j+(\Delta p_i-m_i)(\Delta p_j-m_j)}{W}$。等パワー極限で 00 の $1/(V_i\times V_j)$ に戻る（数値は tools/rotating-team とも突き合わせ済み）。
- 自分の混合比は**相手の**チーム内パワー差だけで動く（たすき掛け依存）。また窓の内側では $|\partial g/\partial\Delta p|<\tfrac12$：チーム内パワー差 1 点は値にして高々 $\tfrac12$ 点で、係数 $1$ の $P$ と対照的。

## 先行研究

- **二体構築から一体選ぶゲームの相性関係**（mathlog）— [theory/refs/20250910.md](../../refs/20250910.md)（ローカルコピー。[原文](https://mathlog.info/articles/JKJaLYZonc2XSVrdaNXL)）。等パワーの構築ベクトル $V$（[00/team_vector.md](../00-single_compatibility_model/team_vector.md) の出典）に加えて、パワー差についても次を部分的に議論している。
  - 2択公式を構築の利得に適用したとき、**分母のパワー項が消える**こと（[two_choice_game.md](two_choice_game.md) の分母補題と同じ観察）。
  - 分子は**パワー2次の項を落とす近似**（パワー差1次まで）で扱い、等パワーでないときは構築同士の関係にパワー差由来の補正ベクトル（記事中の $U$）が付くとしている。$U$ は「パワーで割って補正した相性ベクトル」から $V$ と同じ形で作れる、という書き換えにも触れている。
  - 2択の適用条件（大小大小の交互条件）は**等パワーで代入した形まで**で、パワー差込みの境界は扱っていない。パワーの寄与の詳細（$U$ の影響、パワーの高いエースを補完する相方選びなど）は将来課題として保留されている。
  - 注意：ローカルコピーはマークダウン変換で数式が欠落しており、$U$ の定義式は本文から読み取れない。02 では式を独立に導出し、散文レベルの主張とだけ突き合わせる。
- **二択ゲームのナッシュ均衡**（note）— <https://note.com/pyran19/n/n9e2dc2e577d2>（ローカルコピーなし）。2択公式そのものは [00/two_choice_game.md](../00-single_compatibility_model/two_choice_game.md) が独立に導出済みなので、参照できなくても 02 は進められる。

これに対する 02 の新規部分：[two_choice_game.md](two_choice_game.md) は近似なしの**厳密な**扱いで、パワー2次の双一次項 $(\Delta p_i-m_i)(\Delta p_j-m_j)$ の保持・$P$ の分離・チーム内パワー差の窓（2択 ⇄ 一択の境界）・混合比のパワー依存が先行研究に無い部分。パワー1次で切れば先行研究の近似に整合する。correction_vector.md（未着手）では記事の $U$ との対応を明示する予定。
