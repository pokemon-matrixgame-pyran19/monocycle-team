# 02 - パワー差がある場合の理論

単相性モデル $A_{ij}=p_i-p_j+v_i\times v_j$（[00/model.md](../00-single_compatibility_model/model.md)）を、これまでの**等パワー**（$p$ が相殺する状況）の絞り込みから**パワー差 $p_i-p_j\neq0$ 込み**へ拡張するトピック。model.md の予告「パワー差の補正ベクトルは別途扱う」の回収にあたる。

前提（単相性モデル・2択公式・構築ベクトル $V$）は [00-single_compatibility_model](../00-single_compatibility_model/) を、回転・回転平均は [01-rotating_team](../01-rotating_team/) を参照。

## 中心的な問い

1. **2択ゲームの値のパワー差込み一般化** — 利得 $A_{ij}=p_i-p_j+v_i\times v_j$ の 2×2 ゲームについて、ゲームの値と鞍点（実質一択）条件の場合分けを書き下す。パワー差 $\to 0$ の極限で [00/two_choice_game.md](../00-single_compatibility_model/two_choice_game.md) の結果に戻ることを確認する。
2. **パワー差の補正ベクトル** — パワー差はどんな条件で相性ベクトル側の変換（原点シフトなど）に吸収できるか。吸収できない場合に必要な最小の追加パラメータは何か。
3. **一択化の閾値** — どれだけパワー差があれば相性（外積項）を無視できるか。パワー差と相性項の大小で戦略構造（2択 ⇄ 実質一択）が切り替わる境界を明示する。

コアの3問が閉じたあとの発展として、構築ベクトル $V$ のパワー差込み一般化（二体構築は「1本のベクトル＋α」で表せるか）、01 の回転・回転平均のパワー差1次補正（摂動展開）、パワーと相性の交換レート（パワー1点＝相性どれだけ分か）を予定する。
