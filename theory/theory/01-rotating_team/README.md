# 01 - 相手構築の回転と実質一択

単相性モデルのもとで、自分・相手がそれぞれ2匹で組んだ構築（チーム）を考え、各自1匹を出して戦う2択ゲームの値 $g$ を**実質一択まで込めて**書き下し、片方の構築を固定してもう一方を回転させたとき $g$ が回転角 $\theta$ とともにどう動くか（角度依存とグラフの形）を調べる。

前提（単相性モデル・2択公式・構築ベクトル $V$）は [00-single_compatibility_model](../00-single_compatibility_model/) を参照。

## 中身

- [game_value.md](game_value.md) — 実質一択まで込みの構築同士のゲームの値。鞍点による場合分けと、外積で書いた2択／一択の境界条件。
- [rotation.md](rotation.md) — 相手構築を回したときの配置の縛りと、ゲームの値の角度依存（2択では $\sin\theta$ の逆数、一択では $\sin$ の頭打ち、遷移角と2択の窓）。
- [graph.md](graph.md) — グラフの特徴（折れ目・極小・頭打ち）と対称配置での具体例。
- [rotation_average.md](rotation_average.md) — $\gamma,\delta$ の全大小関係（3パターン＋敵味方入れ替え）での $g(\theta)$ の明示と、一周積分＝回転平均の閉じた式（2択の窓は相殺して平均に効かない）。
- [graph_shapes.md](graph_shapes.md) — P1/P2/P3 それぞれのグラフ形状を言葉で説明し、$S$（縦スケール）・$\eta$（窓の広さ）・$\mu$（傾き＝平均には効かない）の3ノブで「大きくすると形がこう変わる」を整理。

## 要点

- 構築同士のゲームの値は、鞍点（自分の行で最小・相手の列で最大の成分）があればその単体相性 $v_a\times v_b$、なければ混合戦略の $1/(V_i\times V_j)$。
- 「構築は $V$ 1本で決まる」のは2択領域の中だけ。実質一択への切り替わりは $v_1,\dots,v_4$ 個別の位置による。
- 相手を回すと2択領域では $g=1/(R_iR_j\sin\theta)$（$\sin\theta$ の逆数）。発散は手前の一択化でガードされ、対称配置では $|g|\le1$、$V_i\perp V_j$（窓の中心）が最安定。
- 一周の**回転平均**は配置パターンに依らず $\langle g\rangle=\frac{S_iS_j}{2\pi}(\eta_i-\eta_j)$ の1本（長さの和 $S=|v_a|+|v_b|$、**広がり効率** $\eta=|v_a-v_b|/S\in[0,1]$）。符号は $\eta_i-\eta_j$ だけで決まり、長さの積 $S_iS_j$ が勝ち幅を増幅する。2択の窓は上下で相殺して平均に効かない。

## 今後の展開

- ~~$\theta$ を $0$〜$2\pi$ で積分した**回転平均**から、平均的に有利／安定な構築の特徴を引き出す。~~ → [rotation_average.md](rotation_average.md)（全パターンの $g(\theta)$、一周平均の閉じた式 $\langle g\rangle=\frac{1}{2\pi}\big[(|v_3|+|v_4|)|v_1-v_2|-(|v_1|+|v_2|)|v_3-v_4|\big]$、広がり効率 $\eta$ への縮約と配置の比較静学）。
- $v_1,v_2$ の半径・角度を変えた網羅的なグラフ列挙。
- 回転平均と広がり効率 $\eta$ を起点に、**長さ予算 $S_i$ 固定での最適配置**と**構築同士のナッシュ均衡**を攻める。

数値計算・可視化のコードは `tools/rotating-team/` 側で扱う。
