# 01 - 相手構築の回転と実質一択

単相性モデルのもとで、自分・相手がそれぞれ2匹で組んだ構築（チーム）を考え、各自1匹を出して戦う2択ゲームの値 $g$ を**実質一択まで込めて**書き下し、片方の構築を固定してもう一方を回転させたとき $g$ が回転角 $\theta$ とともにどう動くか（角度依存とグラフの形）を調べる。

前提（単相性モデル・2択公式・構築ベクトル $V$）は [00-single_compatibility_model](../00-single_compatibility_model/) を参照。

## 中身

- [game_value.md](game_value.md) — 実質一択まで込みの構築同士のゲームの値。鞍点による場合分けと、外積で書いた2択／一択の境界条件。
- [rotation.md](rotation.md) — 相手構築を回したときの配置の縛りと、ゲームの値の角度依存（2択では $\csc\theta$、一択では $\sin$ の頭打ち、遷移角と2択の窓）。
- [graph.md](graph.md) — グラフの特徴（折れ目・極小・頭打ち）と対称配置での具体例。

## 要点

- 構築同士のゲームの値は、鞍点（自分の行で最小・相手の列で最大の成分）があればその単体相性 $v_a\times v_b$、なければ混合戦略の $1/(V_i\times V_j)$。
- 「構築は $V$ 1本で決まる」のは2択領域の中だけ。実質一択への切り替わりは $v_1,\dots,v_4$ 個別の位置による。
- 相手を回すと2択領域では $g=1/(R_iR_j\sin\theta)$（$\csc$）。発散は手前の一択化でガードされ、対称配置では $|g|\le1$、$V_i\perp V_j$（窓の中心）が最安定。

## 今後の展開

- $\theta$ を $0$〜$2\pi$ で積分した**回転平均**から、平均的に有利／安定な構築の特徴を引き出す。
- $v_1,v_2$ の半径・角度を変えた網羅的なグラフ列挙。
- 回転平均の議論から構築同士のナッシュ均衡を攻める。

数値計算・可視化のコードは `tools/rotating-team/` 側で扱う。
