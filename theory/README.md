# 理論の文章置き場
理論面に関する資料をまとめて管理する。

## ディレクトリ構成
ディレクトリの構成は以下

- idea: アイデアレベルの思いつきやTODO、途中計算など。下書きのファイルを格納。
- theory: 理論の本体として論理の流れや正しい係数の数式などを記したもの。このフォルダの中身が正式な理論の流れになるようにする。
- manuscript: 投稿用の文章の下書き


## 各フォルダの詳細
### theory
- 独立したトピックごとにディレクトリ作ってその中で作業
  - 公開作業は別のフォルダにわけたので公開範囲を意識して無理に区切らなくていい
- ディレクトリに対して連番振る.二桁で０埋めする
- トピック名は英字で単語の区切りはアンダースコア
- つまり"01-topic_name"みたいなフォルダ名

### manuscript
投稿用の文章。作成する時は投稿用のskillに従う。
